# Лабораторная работа 3
## Тема: Исследование алгоритмов классификации

В использованном ранее датасете отсутствует атрибут, который можно было бы выбрать в качестве целевого класса. В связи с этим был подобран
другой датасет, подходящий для решения задачи классификации.
### Описание нового набора данных
+ **Предметная область:** физические характеристики и показатели физической подготовки.  
+ **Источник данных:** https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset
+ **Характер данных:** имитационные.  
+ **Атрибуты:**
  + ***Age*** - возраст члена тренажерного зала (числовой, целочисленный)
  + ***Gender*** - пол посетителя спортзала (категориальный)
  + ***Weight (kg)*** - вес участника в килограммах (числовой, с плавоющей запятой)
  + ***Height (m)*** - рост участника в метрах (числовой, с плавоющей запятой)
  + ***Max_BPM*** - максимальная частота сердечных сокращений (ударов в минуту) во время тренировок (числовой, целочисленный)
  + ***Avg_BPM*** - средняя частота сердечных сокращений во время тренировок (числовой, целочисленный)
  + ***Resting_BPM*** - частота сердечных сокращений в состоянии покоя перед тренировкой (числовой, целочисленный)
  + ***Session_Duration (hours)***	- продолжительность каждой тренировки в часах (числовой, с плавоющей запятой)
  + ***Calories_Burned*** - общее количество калорий, сожженных за каждую тренировку (числовой, с плавоющей запятой)
  + ***Workout_Type*** - вид выполняемой тренировки (категориальный)
  + ***Fat_Percentage*** - процентное содержание жира в организме участника (числовой, с плавоющей запятой)
  + ***Water_Intake (liters)***	- ежедневное потребление воды во время тренировок (числовой, с плавоющей запятой)
  + ***Workout_Frequency (days/week)***	- количество тренировок в неделю (числовой, целочисленный)
  + ***Experience_Level*** - уровень опыта, от начинающего (1) до эксперта (3) (числовой, целочисленный)
  + ***BMI*** - индекс массы тела, рассчитываемый на основе роста и веса (числовой, с плавоющей запятой)
